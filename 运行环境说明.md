# 运行环境说明

## 1. 基本环境

- 操作系统：Windows 10 / Windows 11，Linux 或 macOS 也可运行
- Python 版本：Python 3.10 及以上
- 推荐版本：Python 3.12
- 字符编码：UTF-8
- 是否需要联网：系统运行和评测不需要联网；只有自动下载 LCQMC 数据集时需要联网

本项目使用 Python 标准库实现，不依赖第三方包。

系统提供两种核心模型：

- 规则多特征融合模型：基于 TF-IDF、字符 n-gram、同义词归一化、编辑相似度和极性修正进行人工加权融合。
- 监督学习融合模型：基于 LCQMC 标注数据训练 Logistic Regression，自动学习各特征权重。

## 2. 依赖说明

## 4. 运行方式

以下命令均在项目根目录下执行，例如：

```bash
cd E:\自然语言处理\code
```

### 4.1 启动 Web 原型系统

```bash
python app.py
```

启动后在浏览器中打开：

```text
http://127.0.0.1:8000
```

页面支持输入两个句子，并显示最终相似度、方法对比、子特征分数、否定/反义极性修正结果和分词结果。

### 4.2 命令行计算两个句子的相似度

```bash
python cli.py "我喜欢自然语言处理" "我对NLP很感兴趣"
```

如果需要 JSON 格式输出：

```bash
python cli.py "我喜欢自然语言处理" "我对NLP很感兴趣" --json
```

### 4.3 运行批量示例

```bash
python examples.py
```

### 4.4 Top-K 相似句检索

```bash
python retrieve.py "我对NLP很感兴趣" -k 3
```

系统会从 `data/candidates.txt` 中检索最相似的前 3 个句子。

## 5. 数据集评测

### 5.1 自建小型测试集评测

```bash
python evaluate.py --data data/test_pairs.csv
```

当前自建测试集结果如下：

```text
Accuracy：0.8250
Precision：0.9375
Recall：0.7143
F1：0.8108
```

### 5.2 LCQMC 抽样数据评测

如果已经存在 `data/lcqmc_test_sample.csv`，可以直接运行：

```bash
python evaluate.py --data data/lcqmc_test_sample.csv --threshold 0.40
```

当前 LCQMC 测试集抽样 1000 条的结果如下：

```text
Accuracy：0.5520
Precision：0.5199
Recall：0.9482
F1：0.6716
```

该结果说明：当前轻量级多特征方法召回率较高，但精确率和准确率仍有限，后续可以接入 Sentence-BERT、SimCSE 等预训练语义模型提升深层语义表示能力。

### 5.3 自动搜索阈值

可以在验证集抽样上搜索 F1 最优阈值：

```bash
python evaluate.py --data data/lcqmc_valid_sample.csv --find-threshold
```

项目当前推荐在 LCQMC 抽样实验中使用阈值 `0.40`。

### 5.4 监督学习融合模型

使用 LCQMC 训练集抽样训练 Logistic Regression 融合模型：

```bash
python train_model.py --data data/lcqmc_sample.csv
```

训练完成后会生成：

```text
data/model_weights.json
```

评测监督学习融合模型：

```bash
python evaluate_ml.py --data data/lcqmc_test_sample.csv --threshold 0.16
```

对比规则融合模型和监督学习融合模型：

```bash
python compare_methods.py --data data/lcqmc_test_sample.csv
```

导出监督学习模型误判样例：

```bash
python analyze_errors.py --data data/lcqmc_test_sample.csv --threshold 0.16
```

导出文件：

```text
docs/error_cases.csv
```

当前 LCQMC 测试集抽样 1000 条的对比结果如下：

```text
方法                     Accuracy  Precision   Recall       F1
规则多特征融合              0.5520     0.5199   0.9482   0.6716
监督学习融合                0.6680     0.6119   0.8551   0.7133
```

该结果说明：监督学习融合模型通过标注数据自动学习特征权重，相比人工固定权重，在 Accuracy 和 F1 上有明显提升。

## 6. LCQMC 数据转换

如果已经下载 LCQMC 原始文件，例如：

```text
data/LCQMC.train.data
data/LCQMC.valid.data
data/LCQMC.test.data
```

可以使用以下命令转换为项目评测格式：

```bash
python scripts/convert_lcqmc_local.py data/LCQMC.train.data --sample-size 1000 --output data/lcqmc_sample.csv
python scripts/convert_lcqmc_local.py data/LCQMC.valid.data --sample-size 1000 --output data/lcqmc_valid_sample.csv
python scripts/convert_lcqmc_local.py data/LCQMC.test.data --sample-size 1000 --output data/lcqmc_test_sample.csv
```

转换后的 CSV 格式为：

```text
sentence_a,sentence_b,label
```

其中 `label=1` 表示语义相似，`label=0` 表示语义不相似。

## 7. 单元测试

运行全部单元测试：

```bash
python -m unittest
```

测试内容包括：

- 同义句应得到较高相似度；
- 无关句应得到较低相似度；
- 完全相同句子应得到高相似度；
- 否定极性不一致时应降低分数；
- Top-K 检索应返回最相关候选句。

## 8. 常见问题

### 8.1 运行 LCQMC 自动下载失败

如果运行：

```bash
python scripts/download_lcqmc_sample.py --sample-size 1000 --pool-size 5000
```

出现网络错误，通常是 Hugging Face 在当前网络环境下无法访问。可以改用手动下载 LCQMC 原始文件，再执行本地转换脚本。

### 8.2 找不到 `data/lcqmc_sample.csv`

说明还没有生成 LCQMC 抽样文件。请先运行：

```bash
python scripts/convert_lcqmc_local.py data/LCQMC.train.data --sample-size 1000 --output data/lcqmc_sample.csv
```

### 8.3 中文显示乱码

请确认终端和编辑器使用 UTF-8 编码。Windows PowerShell 可尝试执行：

```powershell
chcp 65001
```
